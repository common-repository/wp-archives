Display your archives with year/month list.

== Description ==

Create a list with all the posts, grouped by months and years like a tree of contents. This plugin is based on [SRG Clean Archives](http://www.geekwithlaptop.com/projects/clean-archives/ "SRG Clean Archives") but modified to be faster and not using additional libraries.

**Features**

*	Easy install
*	Clean list of your posts
*	No additional configuration is required
*	Easy to use

**How to Use**

* Create a page called `archives`.
* On page created to insert the text:

`&lt;!--wp_archives--&gt;`



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `wp-archives` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
